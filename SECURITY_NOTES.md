# Guvenlik Notu

Bu repo herkese acik oldugu icin asagidaki dosyalar bilerek dahil edilmedi:

- canli `site_config.json`
- veritabani kullanici adi ve sifresi
- MariaDB dump dosyalari
- bench icindeki calisan servis durum dosyalari

Yedekte bulunan `bench_meta/site_config.example.json` sadece sablondur.

Acik repo yerine private repo kullanirsan bir sonraki adimda sifreli tam bench arsivi de eklenebilir.
