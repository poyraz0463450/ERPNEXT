# ERPNEXT Yedegi

Bu repo, ARTEGON icin hazirlanan ERPNext calismasinin acik repoya uygun yedegidir.

Icerik:
- `artegon_otomasyon/`: ERPNext uzerine eklenen ozel Frappe uygulamasi
- `reference_erp_ui/`: ornek olarak kullanilan ERP arayuzu ve kaynaklari
- `demo_teknik_resimler/`: sisteme yuklenen demo teknik resim PDF dosyalari
- `bench_meta/`: bench seviyesinde geri yukleme icin gereken guvenli meta dosyalar

Acik repo oldugu icin canli veritabani sifreleri, direkt `site_config.json` ve calisan sistem sirlarini bu yedege bilerek eklemedim.

Tam kurulum notlari icin `RESTORE.md` dosyasina bak.
